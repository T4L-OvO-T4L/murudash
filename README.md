# むるだっしゅ！

この作品は「ムルル」をモチーフにした非公式ファン作品です。  
権利者様とは一切関係ありません。

---

## 🎮 ゲームについて
「むるだっしゅ！」はブラウザ上で遊べる簡単なジャンプアクションゲームです。  
ジャンプボタンでひまわりやブルーベリーをよけまくろう！

---
